## Getting Started

### System Requirements

- **Browser**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Screen Resolution**: Responsive design (mobile, tablet, desktop)
- **Internet Connection**: Stable broadband connection
- **JavaScript**: Must be enabled

### Accessing the Frontend

1. **URL**: Navigate to your frontend URL
2. **Homepage**: View club information and services
3. **Navigation**: Use the main navigation menu

### First-Time Visit

1. **Browse Services**: Explore available facilities and events
2. **Sign In**: Use existing member credentials
3. **Access Dashboard**: View member profile and bookings

## Authentication

### Member Sign In

#### Sign In Process
1. **Access Sign In**
   - Click "Sign In" in the navigation menu
   - Navigate to sign-in page

2. **Enter Credentials**
   - **Email**: Your registered email address
   - **Password**: Your account password

3. **Authentication**
   - Click "Sign In" button
   - System validates credentials
   - Redirects to member dashboard

### Password Management

#### Forgot Password
1. **Access Reset**: Click "Forgot Password?" link on sign-in page
2. **Enter Email**: Provide registered email address
3. **Receive OTP**: Check email for reset OTP
4. **Reset Password**: Enter OTP and new password
5. **Confirm Reset**: Password updated successfully

#### Change Password
1. **Access Profile**: Navigate to dashboard
2. **Security Settings**: Click "Change Password"
3. **Current Password**: Enter current password
4. **New Password**: Enter new password
5. **Confirm Password**: Confirm new password
6. **Update**: Click "Update Password"

## Troubleshooting

### Common Issues

#### Sign In Problems
**Issue**: Cannot sign in to account
**Solutions**:
1. Check email and password
2. Clear browser cache and cookies
3. Try "Forgot Password" option
4. Contact support

#### Booking Issues
**Issue**: Cannot complete booking
**Solutions**:
1. Check facility availability
2. Verify payment method
3. Clear browser cache
4. Try different browser

#### Payment Problems
**Issue**: Payment not processing
**Solutions**:
1. Check payment method details
2. Verify sufficient funds
3. Try different payment method
4. Contact payment support

### Error Messages

#### Common Error Messages
- **"Invalid Credentials"**: Check email and password
- **"Session Expired"**: Re-sign in required
- **"Payment Failed"**: Check payment details
- **"Booking Unavailable"**: Try different time/date

#### Error Resolution
1. **Read Error Message**: Understand the issue
2. **Follow Instructions**: Follow suggested actions
3. **Try Again**: Attempt the action again
4. **Contact Support**: Report persistent issues

### Support Contact

#### Getting Help
- **Email Support**: support@asmc.com
- **Phone Support**: +1-555-0123 (Business hours)
- **Contact Form**: Use website contact form
- **FAQ Section**: Check frequently asked questions

#### Information to Provide
- **Error Details**: Exact error message
- **Steps Taken**: What you were doing
- **Device Information**: Browser, device, OS
- **Account Details**: Email address (for account issues)
